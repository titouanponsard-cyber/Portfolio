Entrepôt de données et agent IA pour les professionnels de santé

Projet réalisé avec Venancio Alves.

Contexte : Les données médicales étaient dispersées sur plusieurs plateformes.

Objectif : Créer une plateforme centralisée permettant la consultation et la recherche avancée d'informations médicales.

Fonctionnalités :

  • Scraping automatisé
  • Nettoyage des données
  • MongoDB
  • Qdrant
  • Recherche sémantique
  • IA Mistral
  • Gestion utilisateurs
  • Authentification
  • Sauvegardes
  • Interface web

Architecture : Scraping → Nettoyage → MongoDB → Qdrant → Flask → Interface Web

Résultat : Base enrichie passant de 1400 à 9000 médicaments.
